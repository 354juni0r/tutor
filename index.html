Cara setting mouse macro FANTECH THOR X9 PUBG Mobile


Ini adalah cara setting mouse macro PUBG Mobile untuk menandai sebuah item drop di PUBG Mobile. Jangan lupa trick ini adalah untuk para pengguna emulator. Oke langsung saja ini langkah-langkahnya :


  1. Kalian download sofware setting mouse macro FANTECH THOR X9 disini
  2. Buka PUBG Mobile (using emulator)
  3. Buka software FANTECH THOR X9
  4. Buka advanced setting
  5. Masukkan tittle klick new
  6. Klick record (masukkan keywoard sesuai yang anda butuhkan)
  7. Stop & save
  8. Go to base setting
  9. Change forward to macro (your script)
  10. Go to PUBG Mobile play training
  11. Input your keywoard to PUBG Mobile di setting keywoard emulator & save
  12. Done
